## üöÄ About the Project

This is my personal developer portfolio designed to showcase my skills, projects, and achievements. It features a modern neon/cyberpunk aesthetic with dynamic 3D elements, smooth scroll animations, and a fully functional contact form.

### ‚ú® Features
- **3D Interactive Elements**: Built using `@react-three/fiber` and `@react-three/drei` (checkout the interactive "Under Construction" page!).

- **Functional Contact Form**: Integrated with `EmailJS` to receive messages directly to my inbox.
- **Responsive Design**: Fully optimized for both desktop and mobile viewing.

## üíª Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/devanshkatkar246/Portfolio.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Environment Variables:**
   Create a `.env` file in the root directory and add your EmailJS configuration to make the contact form work (see `.env.example` for reference):
   ```env
   VITE_EMAILJS_SERVICE_ID=your_service_id
   VITE_EMAILJS_TEMPLATE_ID=your_template_id
   VITE_EMAILJS_PUBLIC_KEY=your_public_key
   ```

5. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in your browser!
